This project is based on the Customer Churn Dataset using Python for data analysis and visualization. The main objective of the project is to analyze customer behavior and identify patterns related to customer churn.
In this project, the dataset is first imported and explored using Pandas and NumPy libraries. Missing values are handled using median replacement techniques, and outliers are detected and removed using the IQR (Inter Quartile Range) method. Categorical data is converted into numerical format using Label Encoding, and numerical features are normalized using Min-Max Scaling.
Different types of data visualizations such as boxplots, histograms, scatter plots, bar charts, pie charts, and line charts are created using Matplotlib and Seaborn libraries to understand customer call patterns and overall data distribution.
The project helps in understanding customer usage behavior, service interaction patterns, and factors affecting churn. It also improves practical knowledge in data preprocessing, feature engineering, data cleaning, and visualization techniques using Python.
